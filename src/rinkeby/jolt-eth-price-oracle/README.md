# JOLT/ETH price oracle

Simple job that works the JOLT/ETH price oracle on Rinkeby.

## Config path

`node_modules/@jolt-network/cli-jobs/dist/rinkeby/jolt-eth-price-oracle`

## Worker requirements

* Must be a valid worker on the Jolt network
